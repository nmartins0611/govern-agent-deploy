USAGE
-----

1. Install Dependencies:
   $ pip install -r requirements-dev.txt

2. Run Tests:
   $ pytest tests/ -v

3. Start Application:
   $ python run.py

4. Test API:
   $ curl http://localhost:5000/api/v1/health

5. Deploy:
   $ /deploy-to-aap
   OR
   $ bash .claude/skills/deploy-to-aap/scripts/deploy.sh
